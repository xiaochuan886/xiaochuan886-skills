# 引用格式规范

## 基本原则

- 优先引用原始政策、官方办事页、法院或法治解读
- 案例要优先引用原始访谈、原始报道或权威媒体
- `research.md` 保留 URL；`article.md` 和 `xhs.md` 不保留 URL
- 涉及政策、法税、注册时，正文中必须保留日期和地区

## 常见引用类型

### 1. 官方政策

正文中引用：

```markdown
根据人力资源社会保障部、税务总局、市场监管总局 2025 年 6 月 23 日发布的相关实施意见，个人创业相关事项正在向“一件事”整合。
```

参考资料格式：

```markdown
1. 人力资源社会保障部办公厅、税务总局办公厅、市场监管总局办公厅. 《关于推动个人创业“高效办成一件事”的实施意见》. 2025-06-23.
```

### 2. 办事指南

正文中引用：

```markdown
北京市 2025 年发布的相关指引显示，AI 与个人创业支持正在进一步对接。
```

参考资料格式：

```markdown
2. 北京市人民政府. 《支持人工智能赋能新型工业化行动方案（2025年）》. 2025-06-05.
```

### 3. 法院或法治解读

正文中引用：

```markdown
最高法 2026 年 3 月的相关文章再次提醒，一人有限责任公司场景下，股东财产独立证明是关键边界。
```

参考资料格式：

```markdown
3. 最高人民法院. 《正确理解公司法第二十三条第一款规定的“不能证明公司财产独立于股东自己的财产”》. 2026-03-07.
```

### 4. 创业案例或媒体报道

正文中引用：

```markdown
据相关创业媒体报道，这类 AI 加持的一人公司开始被当成一个独立创业叙事来观察。
```

参考资料格式：

```markdown
4. [媒体名称]. 《[文章标题]》. [发布日期].
```
