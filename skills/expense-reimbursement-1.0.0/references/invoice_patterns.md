# 发票格式详解

## 高铁电子客票

```
电子发票（铁路电子客票）
发票号码:26329130530000705645
苏州 G7315 杭州西
站 站
2026年03月11日 13:25开 08车05F号 二等座
￥162.00
票价:
3209811990****3776 陆俊峰
购买方名称:山丘生物（杭州）有限公司
```

**关键点**：
- 金额在"票价:"**前面**（￥162.00 在票价:上面）
- 提取：`[¥￥](\d+\.\d{2})`
- 购买方：`购买方名称[：:](\S+)`
- 出行人：`\d{6}\*+\d{4}\s+(\S+)`
- 出行日期：`(\d{4})年(\d{2})月(\d{2})日\s+\d{2}:\d{2}开`

## 打车发票（竖排版）

```
电子发票（普通发票）
购 名称：山丘生物（杭州）有限公司 销 名称：xxx
买 售
方 方
```

**关键点**：
- "购买方"被拆成三行：`购`、`买`、`方`
- 正则 `买\s*方?\s*名称` 匹配不到
- **兜底方案**：`名称[：:]\s*([^\s，。\n]+?有限公司)`，取第一个

## 打车发票（标准版）

```
买 方 名称：山丘生物（杭州）有限公司
```

**提取**：`买\s*方?\s*名称[：:]\s*([^\s，。\n]+)`

## 住宿发票

```
价税合计（大写）柒佰柒拾贰圆陆角肆分（小写） ¥ 772.64
山丘生物（杭州）有限公司 杭州桂圆酒店有限公司
```

**提取**：
- 金额：`（小写）\s*[¥￥]\s*(\d+\.?\d*)`
- 公司：找所有 `xxx有限公司`，**第一个是购买方**

## 行程单格式

```
美团打车 - 行程单
行程人手机号：13618367863
行程数：2笔行程
2026.03.12
1 如祺经济型 杭州 全季酒店 阿里巴巴园区 4.9公里 19.53元
```

**提取**：
- 日期：`(\d{4})\.(\d{2})\.(\d{2})`
- 起点/终点：解析具体格式
- 金额：`(\d+\.?\d*)元`
