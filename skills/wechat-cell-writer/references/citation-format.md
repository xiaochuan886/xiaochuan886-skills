# 引用格式规范

## 基本原则

所有引用的内容必须标明出处，确保信息的可信度和可追溯性。

---

## 引用类型与格式

### 1. 学术论文

**格式**：
```
《期刊名》年份，DOI: xxxxx
```

**正文中引用**：
```markdown
根据《Cell》2025年发表的研究，CAR-T细胞在实体瘤治疗中取得了新突破。
```

**参考文献列表**：
```markdown
1. Wang H, et al. In vivo CAR-T therapy for solid tumors. *Cell*. 2025;182(4):875-889. DOI: 10.1016/j.cell.2025.02.035
```

**常用期刊缩写**：
| 全称 | 缩写 |
|------|------|
| Nature | Nature |
| Cell | Cell |
| The Lancet | Lancet |
| New England Journal of Medicine | NEJM |
| Journal of Clinical Oncology | JCO |
| Science | Science |
| Nature Medicine | Nat Med |
| Clinical Cancer Research | Clin Cancer Res |

### 2. 新闻报道

**格式**：
```
媒体名称 日期
```

**正文中引用**：
```markdown
据丁香园2025年3月1日报道，国内首款TIL疗法已进入临床试验阶段。
```

**参考文献列表**：
```markdown
2. 丁香园. 《国产TIL疗法获批临床，实体瘤治疗迎新选择》. 2025-03-01.
```

**常用医疗媒体**：
- 丁香园 (dxy.com)
- 医学界 (yxj.org.cn)
- 健康界 (cn-healthcare.com)
- 医脉通 (medlive.cn)
- 梅斯医学 (medsci.cn)

### 3. 政策法规

**格式**：
```
发布机构 文号/日期
```

**正文中引用**：
```markdown
根据国家药监局2025年第1号公告，细胞治疗产品临床试验有了新的指导原则。
```

**参考文献列表**：
```markdown
3. 国家药品监督管理局. 《细胞治疗产品临床试验指导原则》. 2025年第1号公告.
```

**常用官方来源**：
| 机构 | 官网 |
|------|------|
| 国家药品监督管理局 (NMPA) | nmpa.gov.cn |
| 国家卫生健康委员会 | nhc.gov.cn |
| 中国药品审评中心 (CDE) | cde.org.cn |

### 4. 公众号文章

**格式**：
```
「公众号名称」《文章标题》
```

**正文中引用**：
```markdown
「悟空说细胞」在文章中详细介绍了NK细胞的工作原理。
```

**参考文献列表**：
```markdown
4. 「悟空说细胞」《NK细胞入门指南：身体的天然免疫卫士》. 2025-02-15.
```

### 5. 临床试验数据

**格式**：
```
试验注册号 + 阶段 + 状态
```

**正文中引用**：
```markdown
一项注册号为NCT0XXXX的II期临床试验显示，TIL疗法的客观缓解率达到35%。
```

**参考文献列表**：
```markdown
5. ClinicalTrials.gov. TIL Therapy in Advanced Melanoma. NCT0XXXX. Phase II.
```

## 正文引用方式

### 直接引用

原文引用他人观点或数据时：

```markdown
《Cell》杂志2025年发表的论文指出："CAR-T细胞能够识别并杀伤表达特定抗原的肿瘤细胞。"[^1]
```

### 间接引用

转述他人观点时：

```markdown
研究表明，NK细胞在抗肿瘤免疫中发挥着重要作用（Wang et al., 2025）。[^2]
```

### 数据引用

引用具体数据时：

```markdown
临床试验数据显示，接受TIL治疗的患者客观缓解率为35%，疾病控制率达66.7%。[^3]
```

## 引用注意事项

### ✅ 正确做法

1. **引用原始来源**：优先引用论文原文，而非转载报道
2. **核实数据准确性**：确保引用的数据与原文一致
3. **注明发布时间**：让读者了解信息的时效性
4. **保留可追溯标识**：在正文或文末保留 DOI、期刊、发布日期等信息；不要插入 URL

### ❌ 错误做法

1. **引用来源不明**：如"网上说"、"据说"
2. **断章取义**：截取部分内容扭曲原意
3. **过时信息**：引用几年前的旧数据作为"最新"
4. **二次引用不标注**：从A文章看到B研究的内容，只标注A

---

## URL 规则

- `research.md` 可以记录 URL、DOI、PMID，便于核查和后续截图
- `article.md` 和最终发布稿 **禁止包含 URL**
- 学术论文优先保留 `期刊名 + 年份 + DOI`
- 新闻、公众号、政策内容保留 `来源名称 + 日期/文号`

## 常见问题

### Q: 网上找不到原始论文怎么办？

A: 可以引用报道该研究的权威媒体，并注明"据XX媒体报道"。

### Q: 数据来自多个来源怎么标注？

A: 可以合并标注：
```markdown
多项临床研究显示，CAR-T治疗血液肿瘤的完全缓解率可达50%-80%[^1][^2][^3]。
```

### Q: 引用了公众号文章，但找不到原文链接？

A: 标注公众号名称和发布日期即可：
```markdown
「XX健康」《文章标题》. 2025-03-01.
```

### Q: 自己公司的临床数据可以引用吗？

A: 可以，但需注明来源：
```markdown
根据XX生物公布的临床数据...
```
或作为公司资料引用。
