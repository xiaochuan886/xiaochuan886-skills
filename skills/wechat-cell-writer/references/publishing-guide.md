# 发布指南

本文档详细说明如何选择主题和颜色发布文章到微信公众号。

## 主题选择

### 根据栏目选择

| 栏目 | 推荐主题 | 推荐颜色 | 理由 |
|------|----------|----------|------|
| **科研前沿解读** | `default` | `blue` | 专业、科技感、权威 |
| **细胞科普系列** | `simple` | `green` | 清新、易懂、健康 |
| **健康管家Tips** | `grace` | `rose` | 温暖、亲切、关怀 |
| **技术分析类** | `modern` | `orange` | 现代、活力、创新 |
| **深度报道类** | `default` | `vermilion` | 严肃、权威、重点突出 |

### 主题详情

| 主题 | 默认颜色 | 风格特点 |
|------|----------|----------|
| `default` | 蓝色 #0F4C81 | 经典风格，二级标题有背景色块，居中对齐 |
| `grace` | 紫色 #92617E | 优雅风格，圆角阴影，文字阴影效果 |
| `modern` | 橙色 #D97757 | 现代风格，大圆角（25px），宽松行距，药丸形标题 |
| `simple` | 绿色 #009874 | 简洁风格，不对称圆角（8px 24px），现代简洁 |

### 主题预览

#### default（经典）
- 二级标题：背景色块 + 白色文字 + 居中
- 三级标题：无特殊样式
- 代码块：Mac 风格三色按钮

#### grace（优雅）
- 二级标题：圆角 8px + 阴影效果
- 三级标题：左侧边框 + 底部虚线
- 文字有轻微阴影

#### modern（现代）
- 容器：大圆角 25px
- 行距：2.0（宽松）
- 二级标题：药丸形状

#### simple（简洁）
- 二级标题：不对称圆角（8px 24px 8px 24px）
- 三级标题：左侧边框 + 浅色背景
- 整体简洁现代

### 颜色覆盖主题默认值

可以通过 `--color` 参数覆盖主题的默认颜色：

```bash
# 使用 modern 主题但用蓝色
--theme modern --color blue

# 使用 grace 主题但用绿色
--theme grace --color green
```

## 发布命令

```bash
npx -y bun ~/.agents/skills/baoyu-post-to-wechat/scripts/wechat-api.ts \
  article.md \
  --theme default \
  --color blue \
  --cover imgs/cover.png \
  --author "细胞小境"
```

### 参数说明

| 参数 | 必需 | 说明 |
|------|------|------|
| `--theme` | 否 | 主题名，默认 `default` |
| `--color` | 否 | 颜色名或十六进制值 |
| `--cover` | 是 | 封面图路径 |
| `--author` | 否 | 作者名，默认 `细胞小境` |

## 发布注意事项

1. **移除外部链接**：微信公众号不允许插入非 `mp.weixin.qq.com` 域名的链接
2. **图片大小**：建议控制在 1MB 以内
3. **参考资料**：使用纯文本格式，不要包含 URL 链接
4. **检查预览**：发布后登录微信公众号后台检查排版效果
