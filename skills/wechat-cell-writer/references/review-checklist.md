# 文章检查清单

发布前的完整检查列表。

## 内容质量

- [ ] 标题吸引人且不违规
- [ ] 篇幅在 1500-3000 字
- [ ] 语言通俗易懂，无专业术语堆砌
- [ ] 无商业植入
- [ ] 逻辑清晰，结构完整

## 引用规范

- [ ] 所有数据有出处
- [ ] 参考资料列在文末
- [ ] 引用格式正确
- [ ] DOI/来源信息可追溯
- [ ] `article.md` 中不包含 URL

## 配图检查

- [ ] 封面图已生成 (900×383px)
- [ ] 配图 1 已生成并插入
- [ ] 配图 2 已生成并插入
- [ ] 论文截图已生成（如引用论文，控制在 1-2 篇核心论文）
- [ ] 图片清晰度达标
- [ ] 图片说明文字完整

## 合规检查

- [ ] 无夸大疗效表述
- [ ] 无绝对化用语 (100%、彻底、根治等)
- [ ] 无误导性内容
- [ ] 无外部链接（微信公众号不允许）
- [ ] 无违禁词

## 发布前最终确认

- [ ] 标题长度合适 (15-30 字)
- [ ] 摘要/引言吸引人
- [ ] 图片位置合理
- [ ] 参考资料格式统一
- [ ] 整体阅读体验流畅

## 常见问题排查

| 问题 | 检查方法 | 修复方案 |
|------|----------|----------|
| 外部链接 | 搜索 `http` | 删除或转为纯文本 |
| 绝对化用语 | 搜索 `100%`、`彻底` | 替换为委婉表述 |
| 图片过大 | 检查文件大小 | 压缩至 1MB 以内 |
| 格式混乱 | 预览 HTML | 检查 Markdown 语法 |
