## 图片命名规范

| 图片 | 文件名 | 说明 |
|------|--------|------|
| 封面图 | `cover.png` | 文章封面，2.35:1 比例 |
| 配图1 | `figure-1.png` | 核心概念图 |
| 配图2 | `figure-2.png` | 数据/流程图 |
| 论文截图 | `paper-1.png` | 核心论文截图 |
| 论文截图 | `paper-2.png` | 第二篇核心论文截图（可选） |

## Frontmatter 字段说明

| 字段 | 必填 | 说明 |
|------|------|------|
| `title` | ✅ | 文章标题 |
| `author` | ✅ | 固定为 `细胞小境` |
| `date` | ✅ | 发布日期 (YYYY-MM-DD) |
| `column` | ✅ | 栏目名称 |
| `tags` | ⚪ | 标签数组 |
| `coverImage` | ✅ | 封面图路径 |
| `images` | ⚪ | 配图路径数组 |
