# 搜索查询示例

信息检索时使用的搜索查询模式。

## 学术研究

```
# 顶级期刊
"细胞治疗" + "最新进展" + 当前年份 + site:nature.com OR site:cell.com
"CAR-T" OR "TIL" OR "NK细胞" + "临床" + "研究" + 当前年份

# PubMed
"cell therapy" + "clinical trial" + 当前年份 + site:pubmed.ncbi.nlm.nih.gov
"CAR-T" + "solid tumor" + site:pubmed.ncbi.nlm.nih.gov
```

## 医疗媒体

```
# 丁香园
"免疫细胞治疗" + "科普" + site:dxy.com
"干细胞存储" + "解读" + site:dxy.com

# 医学界

```


## 搜索技巧

| 技巧 | 示例 | 用途 |
|------|------|------|
| 站内搜索 | `site:nature.com` | 限定来源 |
| 文件类型 | `filetype:pdf` | 搜索 PDF |
| 时间范围 | `after:[近12个月起始日期]` | 限定时间 |
| 精确匹配 | `"CAR-T疗法"` | 精确短语 |
| 排除词 | `-广告` | 排除干扰 |
