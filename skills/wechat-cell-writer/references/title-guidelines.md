# 文章标题写作指南

## 核心原则

### 1. 吸引力原则

标题应该激发读者的阅读兴趣，但必须基于文章真实内容。

**有效技巧**：
- 提出问题：`为什么免疫细胞能"记住"癌细胞？`
- 数字吸引：`CAR-T疗法获批：5年生存率提升60%`
- 对比反差：`小细胞，大作为：NK细胞的抗癌之路`
- 价值承诺：`一文读懂：干细胞存储的5个关键问题`

### 2. 准确性原则

标题必须准确反映文章内容，不做过度承诺。

**正确示例**：
- `最新研究：TIL疗法在晚期肺癌治疗中展现潜力`
- `NK细胞：人体免疫系统的重要成员`

**错误示例**：
- `TIL疗法彻底攻克肺癌！` （夸大）
- `NK细胞：包治百病的神奇细胞` （虚假）

### 3. 合规性原则

遵守微信公众号内容规范和广告法要求。

## 违禁词列表

### 绝对化用语（禁止使用）

| 禁用词 | 可替换为 |
|--------|----------|
| 100% | 研究显示、数据显示 |
| 彻底、完全 | 有效、显著 |
| 根治、治愈 | 治疗、缓解 |
| 必定、一定 | 可能、有望 |
| 最、第一（无依据时） | 领先、重要 |
| 唯一 | 之一、重要 |

### 夸大宣传词（禁止使用）

| 禁用词 | 问题 |
|--------|------|
| 神药、奇迹 | 夸大疗效 |
| 包治、根治 | 虚假宣传 |
| 医生不会告诉你的 | 制造焦虑 |
| 必须存、不存后悔 | 恐吓营销 |

### 标题党词汇（谨慎使用）

| 词汇 | 建议 |
|------|------|
| 震惊！ | 避免使用 |
| 必看！ | 避免使用 |
| 删除前速看 | 避免使用 |
| 疯传 | 慎用 |
| 看完泪奔 | 避免使用 |

## 各栏目标题公式

### 科研前沿解读

```
公式：「最新研究」+ 主题 + 意义
```

**示例**：
- `「最新研究」CAR-T疗法突破实体瘤壁垒，癌症治疗迎来新篇章`
- `「Cell论文解读」科学家发现T细胞"记忆密码"，免疫治疗或迎突破`
- `「临床进展」国产TIL疗法获批临床，晚期实体瘤患者的新选择`

### 细胞科普系列

```
公式：细胞类型 + 核心特征/功能 + 用户价值
```

**示例**：
- `NK细胞：人体内的"天然杀手"，如何守护你的健康？`
- `T细胞：免疫系统的"特种兵"，带你认识它的工作原理`
- `干细胞：身体的"备用零件"，存储它有什么意义？`
- `免疫细胞存储：为未来健康留一把"钥匙"`

### 健康管家Tips

```
公式：问题场景 + 解决方案/答案
```

**示例**：
- `免疫力下降怎么办？这5个方法科学有效`
- `体检发现肿瘤标志物升高，需要恐慌吗？`
- `CAR-T治疗适合哪些人？一文告诉你答案`
- `存储免疫细胞前，你需要了解这4件事`

## 标题优化检查

在确定标题后，请检查以下问题：

### 检查清单

- [ ] 标题是否准确反映文章主要内容？
- [ ] 标题中是否含有违禁词？
- [ ] 标题是否避免了对疗效的夸大？
- [ ] 标题是否避免了绝对化表述？
- [ ] 标题是否避免了制造焦虑或恐慌？
- [ ] 标题长度是否合适（建议15-30字）？
- [ ] 标题是否有吸引力但不做标题党？

### 自检示例

**原标题**：`NK细胞100%杀死癌细胞！存免疫细胞就是存命！`

**问题分析**：
- "100%"是绝对化用语，违规
- "存命"夸大效果，涉嫌虚假宣传
- 整体存在恐吓营销倾向

**修改后**：`NK细胞：人体内的"抗癌卫士"，存储它有什么价值？`

## A/B测试建议

如果有条件，可以准备两个标题进行测试：

**示例**：

| A版（正式） | B版（问答） |
|-------------|-------------|
| NK细胞：免疫系统的"天然杀手" | 什么是NK细胞？带你认识身体的抗癌卫士 |
| CAR-T疗法最新临床进展解读 | CAR-T治疗癌症效果如何？最新数据告诉你 |

根据阅读量和转发率选择更优版本。
