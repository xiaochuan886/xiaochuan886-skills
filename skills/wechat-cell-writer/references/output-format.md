# 信息检索输出格式

信息检索完成后的标准输出格式。

## 标准输出模板

```markdown
## 📊 信息检索结果

### 🔬 学术研究
1. **[Title]** - *Journal* Year
   - 核心发现：...
   - 来源：DOI / URL

2. **[Title]** - *Journal* Year
   - 核心发现：...
   - 来源：DOI / URL

### 📰 行业资讯
1. **[Title]** - 媒体名称 Date
   - 要点：...
   - 来源：URL

2. **[Title]** - 媒体名称 Date
   - 要点：...
   - 来源：URL

### 🏢 同业动态
1. **[公众号名]** 《标题》
   - 角度：...
   - 热度：...

2. **[公众号名]** 《标题》
   - 角度：...
   - 热度：...

### 📋 政策法规
1. **[文件名]** - 发布机构 Date
   - 要点：...

2. **[文件名]** - 发布机构 Date
   - 要点：...
```

## 各类型说明

### 学术研究

优先级最高的信息来源，包括：
- 顶级期刊：Nature, Cell, Science, Lancet, NEJM
- 专业期刊：Nature Medicine, JCO, Clinical Cancer Research
- 预印本：bioRxiv, medRxiv（需标注为预印本）

**必须提取**：
- 论文标题
- 期刊名称
- 发表年份
- 核心发现（1-2 句话概括）
- DOI 或 URL

### 行业资讯

来自权威医疗媒体的报道：
- 丁香园 (dxy.com)
- 医学界 (yxj.org.cn)
- 健康界 (cn-healthcare.com)
- 医脉通 (medlive.cn)
- 梅斯医学 (medsci.cn)

**必须提取**：
- 文章标题
- 媒体名称
- 发布日期
- 核心要点
- 来源 URL

### 同业动态

竞品公众号的文章分析：
- 关注同领域头部账号
- 分析选题角度
- 评估热度（阅读量、点赞数）

**必须提取**：
- 公众号名称
- 文章标题
- 选题角度
- 热度评估

### 政策法规

来自官方机构的信息：
- 国家药监局 (NMPA)
- 国家卫生健康委员会 (NHC)
- 中国药品审评中心 (CDE)

**必须提取**：
- 文件名称
- 发布机构
- 发布日期
- 核心要点

## 检索优先级

| 优先级 | 来源类型 | 权重 |
|--------|----------|------|
| 1 | 学术论文 | 最高 |
| 2 | 官方政策 | 高 |
| 3 | 权威媒体 | 中 |
| 4 | 同业动态 | 参考 |

## 保存位置

检索结果保存到任务文件夹：`{任务文件夹}/research.md`
