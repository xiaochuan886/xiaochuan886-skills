# 内容合规检查清单

本文档用于确保文章内容符合微信公众号平台规范和相关法律法规。

---

## 发布前检查清单

### 一、标题合规检查

- [ ] 是否含有绝对化用语（100%、彻底、根治等）
- [ ] 是否含有夸大宣传词（神药、奇迹、包治等）
- [ ] 是否是标题党（震惊、必看、删除前速看等）
- [ ] 是否准确反映文章内容
- [ ] 是否避免了恐吓营销（不存后悔、错过等死等）
- [ ] 长度是否合适（建议15-30字）

### 二、内容合规检查

- [ ] 是否含有虚假信息
- [ ] 是否夸大疗效或效果
- [ ] 是否使用患者形象代言（未经授权）
- [ ] 是否涉及医疗广告违规内容
- [ ] 是否有商业植入（本skill要求纯科普）
- [ ] 是否引用了来源不明的数据
- [ ] 是否有未标注的引用内容

### 三、用语规范检查

#### 禁止使用的词汇

| 类别 | 禁用词 | 可替换为 |
|------|--------|----------|
| 绝对化 | 100%、彻底、完全、必定、一定、唯一 | 研究显示、数据表明、可能、有望、之一 |
| 疗效夸大 | 根治、治愈、神药、奇迹、特效 | 治疗、缓解、有效、改善 |
| 恐吓营销 | 必须存、不存后悔、错过没机会 | 建议考虑、可以了解、值得思考 |
| 误导性 | 医生不会告诉你、秘密、内幕 | 专家观点、研究进展、新发现 |

#### 需要谨慎使用的表述

| 表述 | 问题 | 建议 |
|------|------|------|
| 最新研究 | 时效性需确认 | 注明发表时间和期刊 |
| 突破性进展 | 可能夸大 | 具体说明进展内容 |
| 首例/首创 | 需核实 | 注明来源和范围 |
| 治愈率/有效率 | 需区分语境 | 区分"临床缓解率"和"治愈" |

### 四、引用规范检查

- [ ] 所有数据是否有出处
- [ ] 引用格式是否正确
- [ ] 参考资料是否列在文末
- [ ] 是否避免引用来源不明的内容
- [ ] 是否避免断章取义

### 五、配图合规检查

- [ ] 图片是否清晰、专业
- [ ] 是否有版权问题
- [ ] 是否与文章内容相关
- [ ] 图片说明是否准确
- [ ] 是否避免使用血腥、恐怖图片

## 常见违规案例

### 案例一：疗效夸大

❌ **违规**：
```
CAR-T疗法100%治愈白血病！存免疫细胞就是救命！
```

✅ **修正**：
```
CAR-T疗法在白血病治疗中展现显著效果，完全缓解率可达50%-80%。
```

### 案例二：恐吓营销

❌ **违规**：
```
35岁后不存免疫细胞，等到得癌就来不及了！
```

✅ **修正**：
```
研究表明，免疫细胞活性随年龄增长而下降。存储免疫细胞可作为健康管理的备选方案。
```

### 案例三：来源不明

❌ **违规**：
```
网上说NK细胞能治百病，很多名人都在做。
```

✅ **修正**：
```
根据《Nature Medicine》2025年发表的研究，NK细胞在多种疾病治疗中展现出潜力。[^1]
```

### 案例四：标题党

❌ **违规**：
```
震惊！这个细胞疗法让晚期癌症患者起死回生！
```

✅ **修正**：
```
晚期癌症患者的新选择：CAR-T细胞治疗的真实案例分享
```

## 敏感话题处理

### 需要特别谨慎的话题

| 话题 | 风险点 | 处理建议 |
|------|--------|----------|
| 癌症治疗 | 疗效宣传、患者案例 | 客观陈述研究数据，避免个案推广 |
| 价格/费用 | 医疗广告嫌疑 | 不提及具体价格，只做科普 |
| 存储推荐 | 商业推广嫌疑 | 本skill要求纯科普，不植入 |
| 临床试验 | 招募嫌疑 | 只介绍进展，不提供联系方式 |
| 对比治疗 | 贬低其他治疗 | 客观介绍各治疗方式特点 |

### 处理原则

1. **科学客观**：基于研究数据，不主观臆断
2. **边界清晰**：科普与广告有明确界限
3. **风险提示**：涉及医疗内容时提醒读者咨询专业医生
4. **免责声明**：必要时添加免责声明

## 免责声明模板

如文章涉及医疗健康内容，可在文末添加：

```markdown
---

⚠️ **声明**：本文仅供科普参考，不构成医疗建议。如有健康问题，请咨询专业医疗机构。
```

## 平台规则参考

### 微信公众号运营规范（摘要）

> 医疗健康类内容应当：
> - 不得含有虚假或夸大内容
> - 不得使用绝对化用语
> - 不得利用患者形象代言
> - 不得含有医疗广告内容
> - 涉及处方药需标注"请按药品说明书或在药师指导下购买和使用"

### 广告法相关规定（摘要）

> 医疗、药品、医疗器械广告不得含有：
> - 表示功效、安全性的断言或保证
> - 说明治愈率或有效率
> - 与其他药品、医疗器械的功效和安全性比较
> - 利用广告代言人作推荐、证明

## 检查通过标准

文章通过合规检查后，应满足：

1. ✅ 标题吸引但不违规
2. ✅ 内容真实可信
3. ✅ 用语规范准确
4. ✅ 引用清晰完整
5. ✅ 无商业植入
6. ✅ 配图专业相关

通过以上检查后，文章方可进入发布流程。
