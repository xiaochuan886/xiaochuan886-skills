# 封面图设计指南

微信公众号封面图的最佳实践和设计规范。

> 本 skill 默认使用 **带中文标题文字** 的公众号封面，而不是纯图封面。

## 尺寸规范

### 标准尺寸

| 类型 | 尺寸 | 比例 | 用途 |
|------|------|------|------|
| 大封面 | 900×383px | 2.35:1 | 首条图文封面（主推） |
| 小封面 | 200×200px | 1:1 | 第2-8条图文封面 |
| 安全区 | 383×383px | 1:1 | 中央区域，关键内容 |

### 安全区原则

> ⚠️ **核心原则**：封面图两侧可能被裁剪，所有关键内容必须放在中央 **383×383px** 安全区内！

```
┌─────────────────────────────────────────────────────────┐
│                    900 × 383px                          │
│  ┌─────────┐                  ┌─────────┐               │
│  │ 裁剪区  │    ┌────────┐    │ 裁剪区  │               │
│  │         │    │ 安全区 │    │         │               │
│  │  可能   │    │383×383 │    │  可能   │               │
│  │  裁剪   │    │        │    │  裁剪   │               │
│  │         │    └────────┘    │         │               │
│  └─────────┘                  └─────────┘               │
└─────────────────────────────────────────────────────────┘
```

## 文字规范

### 标题长度

| 要素 | 最佳长度 | 最大长度 |
|------|----------|----------|
| 主标题 | 8-12字 | 15字 |
| 副标题 | 6-10字 | 12字 |

### 字体选择

| 字体类型 | 推荐字体 | 特点 |
|----------|----------|------|
| 无衬线 | 思源黑体、苹方 | 清晰易读，现代感 |
| 衬线 | 思源宋体 | 传统、权威感 |
| 艺术字 | 适量使用 | 吸引眼球，慎用 |

### 文字颜色

- **白色文字**: 深色背景上使用
- **深色文字**: #1e293b，浅色背景上使用
- **对比度**: 确保文字清晰可读，对比度≥4.5:1

## 六种封面类型

### 1. 数字列表型

**适合**: 科普系列、健康Tips

**设计要点**:
- 大数字突出显示
- 简洁文字说明
- 数字与文字形成视觉层次

**示例**:
```
主标题：5个方法
副标题：科学提升免疫力
视觉：大号"5" + 简洁图标
```

### 2. 问题导向型

**适合**: 科普系列、健康Tips

**设计要点**:
- 用问号吸引注意
- 直击用户痛点
- 简洁的解决方案暗示

**示例**:
```
主标题：免疫力下降？
副标题：这3个信号要警惕
视觉：问号图标 + 警示色
```

### 3. 人物特写型

**适合**: 科研前沿、专家访谈

**设计要点**:
- 真实或插画人物
- 科研/医疗场景
- 增加可信度

**示例**:
```
主标题：最新研究
副标题：TIL疗法新突破
视觉：科研人员 + 显微镜
```

### 4. 对比冲突型

**适合**: 治疗解读、效果展示

**设计要点**:
- 左右或上下对比
- 箭头指示变化
- 明显的视觉差异

**示例**:
```
主标题：治疗前 vs 治疗后
副标题：CAR-T疗法效果
视觉：左右分屏 + 对比箭头
```

### 5. 热点借势型

**适合**: 行业新闻、政策解读

**设计要点**:
- 新闻风格排版
- 权威感设计
- 时间/来源标注

**示例**:
```
主标题：NMPA最新获批
副标题：细胞治疗新进展
视觉：新闻头条风格
```

### 6. 极简文字型

**适合**: 所有栏目，百搭型

**设计要点**:
- 纯文字设计
- 渐变或纯色背景
- 字体排版艺术

**示例**:
```
主标题：NK细胞
副标题：你身体的天然杀手
视觉：大字排版 + 渐变背景
```

## 3秒吸引法则

封面图必须在 **3秒内** 传达三个核心信息：

### 1. What - 文章讲什么
- 标题清晰明确
- 一眼看出主题
- 避免模糊不清

### 2. Why - 为什么要看
- 痛点/问题/好奇
- 价值/收益暗示
- 情感共鸣

### 3. How - 什么风格
- 配色传递情绪
- 图标/图片展示类型
- 整体风格统一

## 质量检查清单

### 设计检查

- [ ] 尺寸正确：900×383px
- [ ] 安全区检查：关键内容在中央383×383px内
- [ ] 标题长度：主标题8-12字
- [ ] 文字清晰：高对比度，可读性强
- [ ] 配色匹配：与栏目风格一致

### 内容检查

- [ ] 主题准确：反映文章核心内容
- [ ] 无误导：标题与内容一致
- [ ] 合规性：无夸大、绝对化用语
- [ ] 品牌一致：符合公众号调性

### 技术检查

- [ ] 文件大小：<500KB
- [ ] 格式：PNG 或 JPG
- [ ] 分辨率：72dpi（网页显示）

## 常见错误

### ❌ 避免的错误

1. **关键内容超出安全区**
   - 问题：文字/图标在两侧被裁剪
   - 解决：所有重要内容放中央383px区域

2. **文字过多**
   - 问题：标题超过15字，难以阅读
   - 解决：精简标题，控制在8-12字

3. **对比度不足**
   - 问题：文字与背景对比度低
   - 解决：使用高对比度配色

4. **风格不统一**
   - 问题：封面与文章内容/风格不匹配
   - 解决：按栏目选择统一配色和风格

5. **过度设计**
   - 问题：元素过多，视觉混乱
   - 解决：简化设计，突出重点
